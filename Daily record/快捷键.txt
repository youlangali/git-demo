Ctrl + D   :    选中光标所在的这个词，继续Ctrl + D选中其他相同的词

Ctrl + U    ：  回退上一个光标操作

Shift + Alt + Left   和  Shift + Alt + Right  扩展/缩小选取范围

Ctrl  + Delete  ： 删除光标右侧的所有字

Shift + End  ：  选择从光标到行尾的所有

Shift  + Home  ： 选择从光标到行首的所有

Ctrl + Shift + [    :   对代码进行压缩

Ctrl + Left   : 光标跳动

Ctrl + Backspace : 删除左侧整个单词

Ctrl + Delet ：删除右侧整个单词

Backspace ： 删除左侧单个字母

Delete : 删除右侧单个字母

Ctrl + Top : 向上滑动页面

Ctrl + Bottom  ：向下滑动页面
